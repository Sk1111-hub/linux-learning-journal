# Scripts

Practice scripts, written while learning. Each script should have a
comment header explaining what it does and why I wrote it.

Example header:

```bash
#!/usr/bin/env bash
# Name: check_failed_logins.sh
# Purpose: Parses auth.log for failed SSH login attempts and counts by IP
# Date: 2026-07-05
# Notes: Written while practicing basic log analysis / brute-force detection
```

Naming convention: `verb_noun.sh` or `verb_noun.py` (e.g. `parse_auth_log.py`).
